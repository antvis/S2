## Pagination

boolean ｜ object **必选**,_default: null_ 功能描述： 分页配置

| 参数      | 说明          | 类型   | 默认值 | 必选  |
| --------- | ------------------- | ------ | ------ | :--:  |
| pageSize  | 每页数量            | `number` | - |  ✓   |
| current   | 当前页（从 1 开始） | `number` |       1      |     |
| total     | 数据总条数          | `number` | - |      |

## NodeField

object 可选，_default：{}_ 功能描述：滚动监听的节点度量

| 参数 | 说明 | 类型 | 默认值 | 必选  |
| --- | --- | --- | --- | :-:  |
| rowField | 行头中需要监听滚动吸顶的度量 id | `string[]` | - |  |
| colField | 列头中需要监听滚动吸「左」的度量 id | `string[]` | - |  |
