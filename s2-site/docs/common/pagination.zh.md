## Pagination

功能描述：分页配置。查看 [文档](/manual/advanced/analysis/pagination) 和 [示例](/examples/react-component/pagination/#pivot)

| 参数      | 说明          | 类型   | 默认值 | 必选  |
| --------- | ------------------- | ------ | ------ | --  |
| pageSize  | 每页数量            | `number` | - |  ✓   |
| current   | 当前页（从 1 开始） | `number` | `1` |  ✓   |
| total     | 数据总条数          | `number` | - |      |
